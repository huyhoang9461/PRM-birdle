# birdle

A five-letter word-guessing game similar to Wordle, built with Flutter.

## Getting Started

If you just cloned this repository, follow these steps to run the project:

1. **Install Flutter**: Ensure you have the Flutter SDK installed on your machine. You can verify your setup by running:

   ```bash
   flutter doctor
   ```

2. **Get Dependencies**: Navigate to the project folder in your terminal and download all the required packages:

   ```bash
   flutter pub get
   ```

3. **Start an Emulator or Connect a Device**:
   - **Android**: Launch an emulator via Android Studio or connect an Android device with USB debugging enabled.
   - **iOS**: Open the Simulator via Xcode (macOS only).
   - **Web/Desktop**: You can also run it on Chrome/Edge or as a native desktop app.

4. **Run the App**: Start the game from your editor (e.g., `F5` in VS Code), or run the following command in your terminal:
   ```bash
   flutter run
   ```
